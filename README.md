# WorkTrip

## サイト概要


### サイトテーマ

地方出張や旅行先で快適に仕事ができる場所を見つけるためのレビューアプリ

---

### テーマを選んだ理由

前職で地方出張を経験する中で、作業できる場所を探す際に情報が複数のサービスに分散しており、効率的な情報収集できないと感じました。

また、Wi-Fiや電源の有無などはレビューに依存しており、確実に把握することが難しいという課題がありました。

この課題を解決するために、作業環境に特化した情報を一元的に管理・共有できるアプリを開発したいと考え、このテーマにしました。

---

### ターゲットユーザ

- 地方出張でリモートワーク・ワーケーションを行う人 
- 旅行先で快適に作業できる場所を探したい人
- Wi-Fiや電源のある作業環境を効率よく探したい人

---

### 主な利用シーン

- 出張先のエリアで快適な作業場所を検索したい時
- レビューを投稿した人の評価やコメントを知りたい時
- 自分が使用した作業場所のレビューを投稿する時
