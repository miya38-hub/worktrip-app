# WorkTrip

## サイト概要


### サイトテーマ

地方出張や旅行先で、快適に仕事ができる場所を見つけるためのレビューアプリ

---

### テーマを選んだ理由

前職で地方出張を経験する中で、作業できる場所を探す際に情報が複数のサービスに分散しており、効率的に情報収集ができないと感じました。

また、Wi-Fiや電源の有無といった作業環境に関する情報はレビューに依存していることが多く、事前に正確に把握することが難しいという課題がありました。

これらの課題を解決するため、作業環境に特化した情報を一元的に管理・共有できるアプリを開発したいと考え、本サービスを制作しました。

---

### ターゲットユーザ

- 地方出張でリモートワーク・ワーケーションを行う人 
- 旅行先で快適に作業できる場所を探したい人
- Wi-Fiや電源のある作業環境を効率よく探したい人

---

### 主な利用シーン

- 出張先や外出先で、快適に作業できる場所を検索したいとき
- 他ユーザーのレビューや評価を参考にしたいとき 
- 実際に利用した作業場所のレビューを投稿したいとき
